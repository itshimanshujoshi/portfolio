## Running Locally

```bash
git clone https://github.com/itshimanshujoshi/portfolio.git
cd portfolio
npm install
```

Create a `.env` file:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

```bash
npm run dev
```
